If the executable complains about missing libraries such as msvcr80.dll, msvcp80.dll or msvcm80.dll you will need to install the
Microsoft Visual C++ C Runtime Libraries.

They can be downloaded here:

Microsoft Visual C++ 2005 SP1 Redistributable Package (x86):
http://www.microsoft.com/downloads/details.aspx?familyid=200B2FD9-AE1A-4A14-984D-389C36F85647&displaylang=en

There is also an older version here, but the above should suffice:

Microsoft Visual C++ 2005 Redistributable Package (x86):
http://www.microsoft.com/downloads/details.aspx?FamilyId=32BC1BEE-A3F9-4C13-9C99-220B62A191EE&displaylang=en
