Structure Synth

Version $VERSION$.

A tool for generating 3D structures by recursive rules.

Created by Mikael Hvidtfeldt Christensen. Licensed and Distributed under the GPL

Acknowledgements:

Structure Synth is my attempt to make a 3D version of CFDG by Chris Coyne, which has been popularized by the Context Free GUI by Mark Lentczner and John Horigan. The EisenScript syntax of Structure Synth was heavily inspired by the CDFG, but has been modified a few places.

The Icons I use are part of the Everaldo: Crystal project.

Structure Synth is built using the Qt cross-platform GUI framework.
